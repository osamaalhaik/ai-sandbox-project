# System Components

## app/api

Contains FastAPI routes and request/response schemas.

## app/core

Contains application configuration, constants, and shared utilities.

## app/sandbox

Contains the Sandbox Manager and process execution logic.

## app/monitoring

Contains process monitoring, syscall tracing, resource sampling, and runtime event collection.

## app/detection

Contains rule-based detection logic and risk scoring.

## app/ai

Contains feature processing, model training, model loading, and anomaly detection.

## app/database

Contains database configuration, sessions, and persistence logic.

## app/models

Contains database models and domain models.

## app/services

Contains orchestration services that connect sandbox execution, monitoring, detection, and storage.

## scripts

Contains operational scripts for running demos, tests, setup tasks, and simulated suspicious behavior.

## data/raw

Contains raw collected behavioral data.

## data/processed

Contains processed features and training-ready datasets.

## dashboard

Contains the future dashboard implementation.

## docs

Contains architecture, design, report material, and presentation notes.

## tests

Contains unit tests, integration tests, and demo validation tests.
